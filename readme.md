# Hermit Crab

The story of a hermit crab travelling the world to find his home shell with his princess.

## Dev Environment

The first time:
```
$ yarn install
```

Then:
- open pico-8
- `cd hermit-crab`
- `load hermit-crab.p8`

Then in your regular terminal:
```
$ yarn build:watch
```

The cart will then be reloaded on save.
